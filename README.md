将图像的exif数据导出到json

# 使用方法

1. **安装 Python 解释器**
2. **配置 Python 环境变量**
3. **clone本项目到本地**
- 下载本项目, 将项目**整个文件夹**丢到图像所在的文件夹下
4. **安装环境依赖**
- 使用命令行工具, `cd` 到项目所在文件夹
- 输入 `pip install -r requirements.txt` 安装依赖

  ```
  pip install -r requirements.txt
  ```

5. **运行**

```
python run.py
```

**运行成功会在项目文件夹下生成一个 `exif.json` 文件**
